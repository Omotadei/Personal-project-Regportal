Task: WebServer using Docker

Docker is a popular container technology for hosting services and applications that virtualizes operating systems and builds lightweight containers. You will create a default base image for this project that other developers can use in their production environments. This project offers the framework for web app images in Python, Ruby, and Meteor.

On successful completion of this project, you will be able to reduce the size of the Dockerfile, shorten the time it takes to write a Dockerfile. Also, you will learn to correctly setup the base system and reduce the amount of time it takes to run docker build and the download time..

Solution:

In this task, the prefered application of interest is a registration portal that collects user data first visit the website and stores the data in a database. The user can then later login with its credentials to on other visits to the website.

Because this task is focused on Docker deployment for devops tasks, major we security would not be taken into account during the course of building this website. there is not need for user passsword to be hashed during registration. The main focus is to successfully deploy the webserver using docker.

To complet this task, the following tools will be used to complet this task:

  1. Python Programming Language
  2. Flask Web Framework
  3. MySQL database
  4. Docker
